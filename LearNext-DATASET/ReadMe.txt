LearNext datasets. Some information about the files format:

1. Pisa:
	-Categories-PISA.txt : Category <space> PoIId <space> Source
	-Pois-PISA.txt : PoIId | City | PoIName | PoILocation (optional) | Latitude | Longitude
	-Trajectories-PISA.tsv : UserId <tab> PoIId;NumberPhotos;StartTime;StopTime <tab> PoIId;NumberPhotos;StartTime;StopTime <tab> etc.

2. Florence
	-Categories-FLORENCE.txt : Category <space> PoIId <space> PoIName <space> Source
	-Pois-FLORENCE.txt : PoIId | City | PoIName | PoILocation (optional) | Latitude | Longitude
	-Trajectories-FLORENCE.tsv : UserId <tab> PoIId;NumberPhotos;StartTime;StopTime <tab> PoIId;NumberPhotos;StartTime;StopTime <tab> etc.

3. Rome
	-Categories-ROME.txt : Category <space> 0 <space> PoIId <space> Source
	-Pois-ROME.txt : PoIId | City | PoIName | PoILocation (optional) | Latitude | Longitude
	-Trajectories-ROME.tsv : UserId <tab> PoIId;NumberPhotos;StartTime;StopTime <tab> PoIId;NumberPhotos;StartTime;StopTime <tab> etc.
